# Waste Sorting Robot

Smart waste sorting system using **YOLO object detection** and an **inductive sensor** to classify and sort metal, plastic, and paper waste.

---

##  Features

* YOLO detects paper & plastic
* Inductive sensor detects metal
* Arduino Nano controls conveyor & pneumatics
* Automated sorting to separate bins

---

##  Tech Stack

**Languages:** Python, C/C++ (Arduino)
**Hardware:** Arduino Nano, Inductive Sensor, Pneumatic Cylinders, Motors, Conveyor
**Software:** YOLO, Python

---

##  How It Works

1. Camera captures waste on conveyor
2. YOLO classifies paper/plastic
3. Inductive sensor detects metal
4. Arduino triggers conveyor & pneumatic sorting
